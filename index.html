Welcome to Net Polling GitHub.

Security issues? No time to monitor your networks? Mobile, PC or tablet, you can see who is using your network through our web interface dynamics. Net Polling allows you to have as a map, one of the shown what is happening on your network, the monitor has never been easier!

Net Polling is an IT project, of semester 5 in systems and networks at IN'TECH INFO, made in python and javascript. The mission was to create a network scanner to have a dynamic map networks permetant to see the bandwidth used.

Finally with Net Polling you can do :

  • see a representation of your network
  • You can manipulate the map with the keyboard or the mouse. You can zoom and d√©zoom the map and move it.
  • Put a machine on listening (Sniffing his network trafic)
  • You can save a copy of the map to show and manipulate it later
  • You can make a finger OS printing and a port scan of the machine of you want

Technologies used for Net Polling:

  • Arp-scan
  • Nmap
  • Tshark
  • Scapy
  • Django
  • Python 2.7
  • Sqlite
  • Javascript
  • Jquery
  • Bootstrap

For the Net Polling server you need :

  • 2 interfaces network, one for the scan network and an other for the mirroring port to analyze the bandwidth
  • A Debian system like You can take a physical server or vmware to do this. With vmware you can do mirroring port

To launch Net Polling after the download, you must do it :

$ sudo python /opt/netpolling/install.py
$ sudo python /opt/netpolling/launch.py
